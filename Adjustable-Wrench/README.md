# Parametric Adjustable Wrench

## Overview
Designed a functional adjustable wrench assembly using Autodesk Fusion 360 to develop skills in parametric modeling, mechanical assemblies, and engineering design.

## Engineering Objectives
- Develop a functional adjustable jaw mechanism
- Create a component-based assembly
- Apply motion joints to simulate real-world movement
- Document the engineering design process

## Skills Demonstrated
- Parametric CAD Modeling
- Mechanical Design
- Component-Based Design
- Mechanical Assemblies
- Motion Joints
- Design Iteration
