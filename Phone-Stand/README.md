# Parametric Phone Stand

## Overview
Designed a parametric phone stand using Autodesk Fusion 360 to develop skills in sketch-based modeling, parametric design, ergonomic product design, and engineering visualization.

## Engineering Objectives
- Design a stable and functional phone stand
- Create the model using parametric sketches and features
- Balance strength, stability, and aesthetics
- Document the engineering design process

## Skills Demonstrated
- Parametric CAD Modeling
- Sketch-Based Design
- Solid Modeling
- Consumer Product Design
- Design Iteration
- Engineering Visualization
- CAD Documentation
