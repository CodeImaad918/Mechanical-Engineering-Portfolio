# Parametric C-Clamp

## Overview
Designed a functional C-clamp assembly using Autodesk Fusion 360 to develop skills in parametric modeling, component-based design, mechanical assemblies, threaded mechanisms, and motion joints.

## Engineering Objectives
- Develop a functional clamping mechanism
- Model the frame, screw, handle, and swivel pad as separate components
- Apply motion joints to demonstrate screw rotation
- Document the engineering design process

## Skills Demonstrated
- Parametric CAD Modeling
- Mechanical Design
- Component-Based Design
- Mechanical Assemblies
- Revolute Joints
- Threaded Mechanisms
- Design Iteration
