# Parametric Screwdriver

## Overview
Designed a parametric screwdriver using Autodesk Fusion 360 to develop skills in solid modeling, parametric design, feature-based modeling, and engineering visualization.

## Engineering Objectives
- Model the screwdriver shaft and handle using parametric features
- Develop a realistic and manufacturable geometry
- Apply engineering dimensions and design intent
- Document the engineering design process

## Skills Demonstrated
- Parametric CAD Modeling
- Solid Modeling
- Feature-Based Design
- Engineering Design
- Sketch Constraints
- Design Intent
- CAD Documentation
